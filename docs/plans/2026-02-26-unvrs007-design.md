## Design System — Liquid Glass

### Palette

| Token | Valore | Uso |
|---|---|---|
| `--bg-deep` | `#04080f` | Background principale |
| `--glass-bg` | `rgba(255,255,255,0.04)` | Pannelli glass |
| `--glass-border` | `rgba(255,255,255,0.08)` | Bordi glass |
| `--glass-blur` | `blur(24px)` | Backdrop filter |
| `--accent` | `#e8e8f0` | Bianco freddo argentato |
| `--accent-glow` | `rgba(180,180,255,0.6)` | Alert, highlight, glow |
| `--accent-dim` | `rgba(232,232,240,0.15)` | Accenti secondari |
| `--text-primary` | `#e8e8f0` | Testo principale |
| `--text-muted` | `rgba(232,232,240,0.45)` | Testo secondario |

### Componenti Glass

- `.glass-panel` — pannelli principali (news, mappa overlay, sidebar)
- `.glass-card` — card dati (conflitti, metriche, paesi)
- `.glass-pill` — badge, tag, indicatori status
- Hover: `border-color` → `rgba(255,255,255,0.2)` + inner glow
- Active/focus: `background` → `rgba(255,255,255,0.08)` + shadow bianco

### Tipografia

- **Display/titoli:** Cormorant Garamond (già presente)
- **UI/corpo:** Geist o Inter — sostituisce Jost
- **Monospace/dati:** Geist Mono — per coordinate, timestamp, valori numerici
